*机器地址信息

切割反馈

1、代理商层级控制该功能，上级有该功能才可给其下级开通该功能。开通后下级代理商机器才会有反馈功能。列表才会有反馈记录流水；

2、若此前开通过已生成反馈记录流水，后续关闭后，反馈记录流水仍然存在。若未生成反馈记录流水，关闭后，反馈记录流水列表菜单隐藏；


机器地址信息

1、代理商层级控制该功能，上级有该功能才可给其下级开通该功能。开通后下级代理商机器列表才会显示新增加的地址等字段；



Cutting feedback

1. The agent level controls the function, and the superior can open the function to its subordinates only if they have this function. After opening, the subordinate agent machine will have the feedback function. The list will have feedback records;

2. If the feedback record flow has been generated through the previous opening, the feedback record flow will still exist after the subsequent closure. If no feedback record flow is generated, the feedback record flow list menu is hidden after it is closed;

Machine address information

1. The agent level controls the function, and the superior can open the function to its subordinates only if they have this function. After the activation is activated, the list of subordinate agent machines will display the newly added address and other fields;
